## Quick Reference

| Scenario | Expected Behavior |
|----------|------------------|
| **First visit** | No cookies, 401 expected |
| **After login** | Cookies set, 200 OK |
| **Token expires** | Auto-refresh, retry |
| **Refresh fails** | User must login again |
| **CORS blocked** | No cookies sent, 401 |
| **Wrong domain** | Cookies not sent, 401 |
